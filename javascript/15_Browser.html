# 자바스크립트로 웹브라우저 직접 제어하기

  • 자바스크립트는 웹브라우저와 소통하기 위한 객체(BOM)을 통해 웹브라우저를 직접 제어할 수 있다.
  • window 객체는 사용자가 현재 사용중인 웹브라우저를 의미한다.
  • window 객체는 생략이 가능하다. (원래는 document 도 window.document이다.)

# Browser Object Model

  • window : 웹브라우저 최상위 객체
  • window.screen : 사용자의 모니터 정보가 들어 있는 객체
  • window.history : 사용자의 웹브라우저의 페이지 이동 내역 객체
  • document : 현재 보고 있는 페이지의 Document Object Model
  • window.location : 현재 보고 있는 위치(url)에 대한 정보를 담은 객체
  • window.navigator : 웹브라우저의 정보를 담은 객체
