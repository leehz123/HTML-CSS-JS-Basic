# 자바스크립트

  • 웹페이지를 직접 제어할 수 있는 프로그래밍 언어
  • 웹브라우저에 자바스크립트를 해석할 수 있는 자바스크립트 엔진(인터프리터?)이 내장되어 있다.
  • (자스가 컴파일도 진행하고 인터프리터도 하기 때문에 백퍼 인터프리터언어라고 할 순 없음)
  • 웹 점유율 100% (js 없이 웹프로그래밍 하기 불가능하단 뜻. 대체가 없음. 요새 각광받는 리액트나 뷰도 따지고 보면 자바스크립트)

# 웹페이지에 자바스크립트를 적용하는 3가지 방식

  • 특정 요소의 이벤트 속성에 사용
  • script태그를 사용
  • 다른 자바스크립트 파일을 가져와서 사용
