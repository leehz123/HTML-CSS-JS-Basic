# 반응형 웹페이지

  • viewport의 크기에 따라 스타일을 다르게 적용하는 페이지
  • viewport : 현재 웹페이지를 보고 있는 프로그램

# media query

  • all : 모든 기기에 적용되는 미디어 쿼리
  • screen : 스크린 기기(일반 화면)에 적용되는 미디어 쿼리
  • speech : 대신 읽어주는 프로그램에 적용되는 미디어 쿼리
  • print : 인쇄 결과물에 적용되는 미디어 쿼리
    • 대충 all 또는 screen쓰면 됨
