# display property

  • 해당 요소가 어떻게 보여야 할지 결정하는 속성
  • 기본값으로는 inline과 block이 있다

# inline 요소의 특징(display : inline)

  • 문단 내부에서 영역을 지정할 때 사용하는 태그들
  • 새 줄에서 시작하지 않고, 해당 요소의 내용만큼 영역을 차지한다
  • padding과 margin을 설정하더라도 고려하지 않는다.
  • 잉 설명이 다른데? 코드잇에서는 inline레벨일 때 margin-top과 marginbottom은 적용되지 않지만,
    padding은 이와 상관 없이 상하좌우 다 된대
  • 코드잇 설명 주소
  • ex> span, b, i(이탤릭), ...

# block 요소의 특징 (display : block)

  • 항상 새 줄에서 시작한다
  • 기본적으로 한 줄 전체를 차지한다
  • padding과 margin을 제대로 적용할 수 있다
  • ex> div, p, h1, li, ul, ...

  • 문단 내에서 글자 취급 받지만 padidng과 margin을 제대로 적용할 수 있는 display
  • inline과 block의 단점을 보완하였다.
