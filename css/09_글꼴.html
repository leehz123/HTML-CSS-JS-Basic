# font-family

  • font-family 속성으로 원하는 글꼴을 설정할 수 있다.
  • 웹페이지를 보는 사용자의 컴퓨터에 설치되지 않은 글꼴은 적용되지 않는다.
  • 글꼴이 없는 경우를 대비하여 여러 예비용 글꼴을 설정해 놓을 수 있다.
  • 사용자의 컴퓨터에 글꼴을 설치하는 경우보다 Font 관련 CDN을 사용하는 경우가 많다.
  • CDN (Content Delivery Network) : 로컬 자원이 아닌 다른 원격 자원을 웹페이지 구성에 사용하는 것
