# 요소의 위치 마음대로 설정하기

  • position 속성을 통해 요소의 위치를 마음대로 설정할 수 있다.
  • positon 속성을 사용하기 위해서는 부모의 요소의 포지션 속성에 기본값을 제외한 다른 포지션 값이 설정 돼 있어야 한다.
  • 부모요소에 포지션이 설정 돼 있지 않다면 가장 가까운 포지션이 설정된 부모 요소를 기준으로 위치가 설정된다.

# 포지션의 종류

  • static : 기본값, 포지션을 사용하지 않는다.
  • absolute : 절대 위치를 설정한다.
  • relative : 요소의 원래 위치에서 상대적인 위치를 설정한다.
  • fixed : 웹 브라우저에서 고정적인 위치를 차지한다.
  • sticky : 스크롤이 내려갈 때 따라 움직이도록 설정한다.
