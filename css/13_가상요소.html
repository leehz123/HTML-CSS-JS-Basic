# Pseudo Element

  • 해당 요소의 특정 부분을 선택하는 것. 선택자 뒤에 ::을 붙여서 원하는 가상 요소를 선택한다.
  • ::first-letter - 첫 번째 글자
  • ::first-line - 첫 번째 줄
  • ::selection - 블록 지정으로 선택된 부분
  • ::before - 요소 시작 전
  • ::after - 요소 끝난 후
