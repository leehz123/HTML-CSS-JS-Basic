# 절대 길이 단위

  • px : 픽셀(화소)
  • mm : 밀리미터
  • cm : 센치미터
  • in : 인치
  • 폰트사이즈 뿐만 아니라 px이 들어갔던 자리들 (width, height ...)에도 먹힘

# 상대 길이 단위

  • em : 현재 요소에 적용된 font-size에 비례한 크기
    ex> 2.0em일 때 현재 요소에 적용된 font-size의 2배 크기 (2글자 정도 들어갈 공간을 확보하고 싶을 때)
  • rem : 현재 페이지의 전체(html태그)에 적용된 font-size에 비례한 크기 조절
    ex. 2.0rem일 때 현재 페이지 전체 font-size의 2배 크기 (얘는 요소의 글자 크기가 아닌 html태그에 적용된 글자크기를 기준으로 한다는 것!)
  • vw: viewport width. 현재 웹 페이지를 보고 있는 프로그램(viewport)의 너비에 비례한 크기 조절
    ex> 1vw는 viewport width의 1% 크기
  • vh: viewport height. viewport의 높이에 비례한 크기 조절
    ex> 1vh는 viewport height의 1% 크기
