# table tag

- <table> : 표 전체를 나타내는 태그
- <tr> : table row. 해당 테이블에 한 행을 추가한다.
- <td> : table data. 해당 행에 일반 칸을 추가한다.
- <th> : table header. 해당 행에 제목 칸을 추가한다.
