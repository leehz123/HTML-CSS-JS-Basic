# HTML 요소

<tagname>요소의 내용(Contents)</tagname>

- 모든 요소는 여는 태그와 닫는 태그(슬래시가 붙어 있는 태그)로 구성된다.
- 여는 태그, 닫는 태그, 해당 요소의 내용을 모두 합쳐 요소(Element)라고 부른다.
- 태그의 이름에 따라 웹브라우저에게 다른 방식으로 해석된다.

#HTML의 기본 요소들

!DOCTYPE html : 이 문서가 html문서임을 표시하는 태그 html : 해당 태그의 범위가 HTML페이지 전체를 나타낸다. head : 이 HTML페이지의 보이지 않는 설정 부분을 담당하는 태그.
body : 이 HTMl 페이지의 눈에 보이는 부분을 담당하는 태그(내용) _ 모든 내용은 body태그 안에 들어가야됨
